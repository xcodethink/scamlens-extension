AI 智能书签管理器：自动摘要、死链检测、页面快照、语义搜索、诈骗防护。免费使用。

## 详细描述

ScamLens - Smart Bookmarks 是一款强大的 Chrome 书签管理器，借助 AI 自动整理、保护和增强您的网络收藏。

🤖 AI 智能整理
一键保存任何页面，或使用 Ctrl+Shift+S 快捷键。AI 即时生成摘要和标签，告别手动分类。

📸 页面快照 — 永不丢失内容
提供三级页面快照：轻量文本（~10KB）、带格式文本（~100KB）、含图片完整存档（~1MB）。即使网站下线，您保存的内容仍可在内置阅读器中随时访问。

🔗 死链检测
自动健康检查按您选择的频率（每日、每周或每月）扫描书签中的失效链接。双重确认机制有效避免临时故障导致的误报。

🔍 语义搜索
通过含义而非关键词查找书签。基于向量的搜索理解上下文 — 搜索"机器学习教程"也能找到标题不含这些词的 AI 课程书签。

🛡️ 域名情报与诈骗防护
每个收藏的域名自动通过 WHOIS/RDAP 分析注册日期、注册商、DNS 服务器和 DNSSEC 状态。一眼识别可疑的新注册或配置异常的网站。完整诈骗报告请访问 scamlens.org。

🔐 隐私优先
• 所有书签本地存储 — 除非您选择云备份，数据绝不离开您的设备
• 可选端到端加密（E2EE）跨设备云同步
• 无跟踪、无广告、不出售数据
• 透明隐私政策：https://scamlens.org/privacy

🔑 灵活的 AI — 支持自带密钥
使用内置免费额度，或接入您自己的 API 密钥：支持 Anthropic Claude、OpenAI、Google Gemini、Grok 及任何 OpenAI 兼容服务商。

📋 更多功能
• 新标签页面板 — 用精美的书签管理器替换默认新标签页
• 重复检测 — 智能查找并合并重复书签
• 批量标签编辑 — 几秒内整理数百个书签
• 导入/导出 — 支持 Chrome 书签导入和 JSON 备份
• 右键保存 — 右键菜单快速收藏
• 快捷键 — Ctrl+Shift+S 保存，Ctrl+Shift+B 打开管理器

💰 套餐
• 免费版：每月 15 次 AI 操作，无限书签，本地存储
• 专业版（$4.99/月）：200 次 AI 操作，E2EE 云同步，云备份
• 强力版（$9.99/月）：无限 AI，支持自带 API 密钥（BYOK）

🌐 多语言支持
中文、English、Tiếng Việt、Español、Français、Deutsch、日本語、한국어、Português

支持 Chrome 102+ · Manifest V3 · 公开隐私政策
